Appendix A: List of All Recognized Symbols and Tokens

Token Description Recognized Symbol(s)
TIdent Identifier (variable or function name) A letter followed by zero or more letters, numbers, or underscores (e.g. x Fn example_ident p2)
TFloatLit Literal decimal number and/or scientific notation Any positive number (Note: negation is treated operation) with a decimal point or 'e' somewhere in it (e.g. 0.3 .125 3.14 6.02e23 1.602E-19)
TIntLit Integer One or more decimal digits (e.g. 1234 095 42 7)
TConstant An identifier that has a constant value (number, set, or truth value) Usually preceded by a # sign, but some keywords are recognized. See the list of constants below.
TPlus Addition +
TMinus Subtraction -
TPlusMinus "Plus or Minus" &pm +/-
TMinusPlus "Minus or Plus" &mp -/+
TTimes Multiplication *
TDivide Division (vertical notation; \( \tfrac{a}{b} \)) /
TSlash Division (horizontal notation; \( a / b \)) &/
TExponent Exponentiation/Power ^ **
TRatio Ratio ("the ratio of a to b") ::
TRatioEqual Equal Relation Operator for ratios ("(a :: b) as (c :: d)" reads "a is to b as c is to d") as
TModulus Modular Arithmetic Operator mod %
TBang Exclamation Point (used for negation and factorial) !
TReal Real Part Filter Operator &Re
TImaginary Imaginary Part Filter Operator &Im
TEqual "Equal To" Relation Operator (for algebraic expressions) = ==
TNotEqual "Not Equal To" Relation Operator (for algebraic expressions) != /= <>
TLess "Less Than" Relation Operator <
TLessEqual "Less Than or Equal" Relation Operator <=
TGreater "Greater Than" Relation Operator >
TGreaterEqual "Greater Than or Equal To" Relation Operator >=
TDivides Divisibility Relation Operator divides
TNotDivides Negation of Divisibility Relation !| ndivides ndivide notdivides notdivide
TCongruent "Congruent" Relation Operator ~= congruent
TSimilar "Similar To" Relation Operator (sometimes also TTilde depending on context) similar sim
TParallel "Parallel To" Relation Operator parallel para
TPerpendicular "Perpendicular To" Relation Operator perpendicular perp
TSubset Subset Relation Operator subset
TPropSubset Proper Subset Relation Operator propersubset propsubset psubset
TSuperset Superset Relation Operator superset supset
TPropSuperset Proper Superset Relation Operator propersuperset propersupset propsuperset propsupset psuperset psupset
TIn Inclusion Relation Operator in
TPipe Vertical Bar Character |
TUnion Set Union Operator union
TIntersect Set Intersection Operator intersect
TSetDiff Set Difference Operator \
TDirectSum Direct Sum Operator &o+
TCartesianProduct Cartesian Product Operator &*
TEquiv "Equivalent" Relation Operator === equiv
TNotEquiv "Not Equivalent" Relation Operator !== /== nequiv
TQForall Universal Quantification ("for all") forall
TQExists Existential Quantification ("there exists") exists
TQUnique Unique Quantification ("there exists a unique") unique
TColon Colon Character ("such that") :
TIff Biconditional Logical Operator ("if and only if") <-> iff
TImplies Conditional Logical Operator ("implies", "if-then") -> implies onlyif
TIf Reverse Conditional Logical Operator ("implied by") <- if when whenever impliedby
TThen Keyword used for parsing "if-then" implication then
TAnd Logical Conjunction and &&
TOr Logical Disjunction or ||
TXor Logical Exclusive-Or xor
TNot Logical Negation (sometimes also TTilde or TBang depending on context) not
TTilde Tilde Character (used for negation and similarity) ~
TCompose Function Composition Operator @
TPrime Prime Differentiation '
TDotDiff Dot Differentiation
TChangeDelta Difference Operator &D
TDifferential Differential "D" &d
TPartial Partial Differential "D" &pd
TIntegral Integration Symbol integral Integral int Int &integral &Integral &int &Int
TVectorizer Arrow over a vector variable &v
TUnitVectorizer "Hat" over a unit vector variable &u
TDot Dot Product Operator &.
TCross Cross Product Operator &x
TWedge Wedge Product Operator &w
TTensor Tensor Product Operator &ox
TGradient Gradient Operator ("del" or "grad") &del &grad
TDivergence Divergence Operator ("div") &del. &div
TCurl Curl Operator ("curl") &delx &curl
TLParen Left (Open) Parenthesis (
TRParen Right (Close) Parenthesis )
TLCurlyBrace Left (Open) Curly Brace {
TRCurlyBrace Right (Close) Curly Brace }
TLSqBracket Left (Open) Square Bracket [
TRSqBracket Right (Close) Square Bracket ]
TLRangeInclusive Left Delimiter for Inclusive Range [:
TRRangeInclusive Right Delimiter for Inclusive Range :]
TLRangeExclusive Left Delimiter for Exclusive Range (:
TRRangeExclusive Right Delimiter for Exclusive Range :)
TLPipe Left Pipe Delimiter |:
TRPipe Right Pipe Delimiter :|
TLDoublePipe Left Double-Pipe ("Norm") Delimiter ||:
TRDoublePipe Right Double-Pipe ("Norm") Delimiter :||
TLVector Left Angle Bracket <:
TRVector Right Angle Bracket :>
TComma Comma separator ,
TSuperscript Superscript Index Selector &^
TSubscript Subscript Index Selector &_
TSemicolon Semi-Colon Character ;
