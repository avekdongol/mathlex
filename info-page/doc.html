Documentation

NOTE: MathLex is distributed in a precompiled, optimized, and minified file. Building MathLex from source should not be necessary unless you know what you are doing and need/want to make changes.

Make sure you have the following applications and libraries installed:

  • Node.js with the following modules: CofeeScript, Handlebars, Jison, and UglifyJS
  • Ruby with the Compass gem.

Run make from your console to run the automated build script:

% cd /path/to/mathlex-source
% make

The compiled JS files will be written to build/browser
